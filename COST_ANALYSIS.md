# Gemini 3 Flash コスト分析

## 📊 正確な料金体系

### Gemini 3 Flash Preview (`gemini-3-flash-preview`)
- **Input:** $0.50 per 1M tokens ($0.0005 per 1K tokens) ※テキスト
- **Output:** $3.00 per 1M tokens ($0.0030 per 1K tokens)

> モデルは `api/config.ts` の `MODEL_NAME` でバージョン固定しています。
> エイリアス（`gemini-flash-latest`）は指す先と単価が予告なく変わるため使いません。
> Preview版のため、GA時にモデルIDが `gemini-3-flash` へ変わる可能性があります。

## 💰 典型的なリクエストのコスト計算

### シナリオ例
- **ユーザーメッセージ:** 200文字
- **会話履歴:** 5メッセージ (約500文字)
- **合計Input:** 700文字 ≈ 1,050トークン
- **AI応答Output:** 500トークン

### コスト計算
```
Input Cost  = 1,050 tokens × $0.0005 / 1K = $0.000525
Output Cost = 500 tokens × $0.0030 / 1K  = $0.0015
────────────────────────────────────────────────────
Total per request = $0.002025 (約 $0.0020)
```

## 🚨 重要な注意事項

### 1. 無料枠の活用
Gemini APIには無料枠があります：
- Input/Output: 無料（制限あり）
- 適切に活用すれば初期コストは最小限

### 2. コスト最適化の推奨事項

#### 即効性のある対策
1. **会話履歴の制限** (現在: 10メッセージ・1件8000文字上限)
   ```typescript
   const recentHistory = conversationHistory.slice(-5); // 10 → 5に削減
   ```
   効果: Input トークン ~50%削減

2. **出力トークン数の最適化** (現在: 4096)
   ```typescript
   maxOutputTokens: 2048, // 4096 → 2048に削減
   ```
   効果: 出力コスト最大50%削減

3. **レート制限の調整（さらに絞る場合）**
   ```typescript
   global: {
     maxCostPerHour: 3.0,  // $5 → $3
     maxCostPerDay: 5.0,   // $10 → $5
   }
   ```
